# Phase 2 ‚Äî UI Design Contract

> Visual and interaction contract for the admin supply-onboarding console. Generated by gsd-ui-researcher, verified by gsd-ui-checker.
>
> **Inheritance:** Phase 1 established the locked platform design system (PLAT-03). This spec reuses those tokens verbatim and adds only the NEW surfaces Phase 2 introduces ‚Äî forms, lists/tables, cards, and the "you keep" calculation panel. Do not re-derive tokens; consume them.

**Existing reusable primitives (do NOT rebuild):**
- `platform/ui/Button.tsx` ‚Äî 52px teal CTA, `min-h-[44px]` hit target, `text-[16px]`/`font-semibold`. Phase 2 primary actions use this as-is.
- `platform/ui/StatusDot.tsx` ‚Äî coloured dot + text label; for Phase 2, reuse the dot+label pattern for the destination `active`/`inactive` indicator (see Color).
- `platform/ui/LanguageToggle.tsx` ‚Äî EN/BG toggle.
- `platform/i18n/{en,bg}.ts` ‚Äî dictionary; ALL new Phase 2 copy below must be added as keys here (PLAT-04). EN values are listed in the Copywriting Contract.

**New primitives Phase 2 must build (first phase to need them ‚Äî keep platform-generic, unprefixed):**
- `TextField` / `Input` (label + input + inline error slot)
- `Select` (for company/property parent pickers, airport, zone)

- `Card` (entity summary container)
- A secondary/ghost button variant (deferred in Phase 1 D-08; needed here for Edit/Cancel) ‚Äî extend `Button` with a `variant` prop rather than a new component.

## Spacing Scale

Declared values (multiples of 4 only ‚Äî matches the px values already hard-coded in Phase 1 primitives):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Dot‚Üîlabel gap, inline icon gaps, tight label/help-text gap |
| sm | 8px | Input vertical rhythm, chip padding, table cell compact padding |
| md | 16px | Default gap between form fields; body padding; CTA `text-[16px]` baseline |
| lg | 24px | Button horizontal padding (`px-[24px]`, locked); card padding; form section gap |
| xl | 32px | Gap between form sections; page content top padding |
| 2xl | 48px | Major console section breaks (e.g. list ‚Üî create form) |
| 3xl | 64px | Page-level outer spacing on desktop console |

Exceptions: CTA height is a fixed **52px** (brand-locked, not on the 4-scale) with a **44px** minimum hit target floor (WCAG 2.5.5). All other interactive controls (inputs, selects, secondary buttons, table-row tap targets) must be **‚â•44px** tall.

## Typography

Reuse the Phase 1 roles exactly. Four sizes, two weights (regular 400 + semibold 600). No new sizes.

| Role | Size | Weight | Line Height | Usage in Phase 2 |
|------|------|--------|-------------|------------------|
| Body | 16px | 400 | 1.5 | Field values, help text, table cell text, "you keep" line items |
| Label | 14px | 600 | 1.4 | Field labels, table column headers, status labels, the StatusDot label |
| Heading | 20px | 600 | 1.2 | Card titles, form section headings ("Company details", "Pricing") |
| Display | 28px | 600 | 1.2 | Page title ("Companies", "Invite a driver") |

Notes:
- Weights are strictly **400** and **600** ‚Äî no 500/700.
- The "you keep (before fees)" amount is emphasised by weight (600) and the `slate` colour, NOT by introducing a new size or the accent colour.
- Body line-height 1.5; heading/display 1.2 ‚Äî already encoded in StatusDot (`leading-[1.4]` for the 14px label).

## Color

The locked six-colour-plus-white palette (`app/globals.css` `@theme`). Phase 2 maps it to the 60/30/10 model for the **admin desktop console** surface.

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | `#ffffff` white | Console page background, form surface, table background |
| Secondary (30%) | `#2f4858` slate | Sidebar / top nav, page headings & primary text, dark surface chrome |
| Accent (10%) | `#029b87` teal | See reserved-for list below |
| Destructive | `#e44b4b` coral | Deactivate/delete confirmations, validation errors, "breaks shared links" slug warning |

Supporting brand tokens used semantically (NOT as additional accents):
- `#047982` teal2 ‚Äî `paid`/secondary states (not used as an interactive accent in Phase 2).
- `#febe21` amber ‚Äî attention/in-progress (not a Phase 2 interactive accent).
- `#66676f` grey ‚Äî muted/help text, the `inactive` destination indicator dot, `requested`/`completed` states.

**Accent (teal) reserved for:**
- Primary CTA fill (`Button`: "Create company", "Save destination", "Invite driver", "Save", "Generate invite link").
- Active/focused form-field border + focus-visible outline.
- Text links and the selected item in the company‚Üíproperty‚Üídestination navigation.
- The active-row / active-tab indicator in the console list.

Accent is NOT used for: every button (secondary actions use a ghost/outline slate variant), table borders, body text, or decorative fills.

**Status indicators (reuse StatusDot pattern ‚Äî colour is never the sole signal, WCAG 1.4.1):**
- Destination `active` ‚Üí teal dot + "Active" label.
- Destination/property/company `inactive` ‚Üí grey dot + "Inactive" label.
- "Cannot deactivate ‚Äî has active children" is communicated by coral helper TEXT, not colour alone.

## Copywriting Contract

All copy below is added to `platform/i18n/en.ts` (and translated in `bg.ts`) as new keys ‚Äî never hard-coded in JSX (PLAT-04). EN canonical values:

| Element | Copy |
|---------|------|
| Primary CTA ‚Äî company | `Create company` (edit mode: `Save changes`) |
| Primary CTA ‚Äî property | `Add property` (edit mode: `Save changes`) |
| Primary CTA ‚Äî destination | `Save destination` |
| Primary CTA ‚Äî driver invite | `Generate invite link` |
| Secondary CTA | `Cancel` |
| Empty state heading (companies) | `No companies yet` |
| Empty state body (companies) | `Create your first company to start adding properties and bookable destinations.` |
| Empty state heading (properties) | `No properties yet` |
| Empty state body (properties) | `Add a property under this company. Each property can have one or more bookable destinations.` |
| Empty state heading (destinations) | `No destinations yet` |
| Empty state body (destinations) | `Add a destination to generate its shareable /pickup link and set its price.` |
| Empty state heading (drivers) | `No drivers invited yet` |
| Empty state body (drivers) | `Invite a driver to build your claim pool. Drivers set their own password from the link you share.` |
| Error ‚Äî required field | `This field is required.` |
| Error ‚Äî slug taken | `That link is already in use. Choose a different one.` |
| Error ‚Äî slug invalid | `Use lowercase letters, numbers, and hyphens only.` |
| Error ‚Äî commission range | `Commission must be between 0 and 100%.` |
| Error ‚Äî save failed (generic) | `Couldn't save. Check your connection and try again.` |
| "You keep" ‚Äî commission line | `Company commission ({pct}%): ‚Ç¨{amount}` |
| "You keep" ‚Äî net line | `You keep (before fees): ‚Ç¨{amount}` |
| "You keep" ‚Äî fee note | `Estimated Stripe fee ~1.5% + ‚Ç¨0.25 per booking (applied at payment, Phase 3).` |
| Slug edit warning (D-10) | `Changing this link will break any /pickup links you've already shared. The old link will stop working.` |
| Invite link delivery note (D-04) | `Copy this link and send it to the driver. They'll set their own password and sign in.` |
| Destructive ‚Äî deactivate destination | `Deactivate destination?`: `Its /pickup link will stop working immediately. You can reactivate it later.` |
| Destructive ‚Äî deactivate property | `Deactivate property?`: `Deactivate its destinations first. A property with active destinations can't be deactivated.` |
| Destructive ‚Äî deactivate company | `Deactivate company?`: `Deactivate its properties first. A company with active properties can't be deactivated.` |
| Destructive ‚Äî hard delete | `Delete permanently?`: `This can't be undone. Only available because nothing references this record.` |
| Destructive confirm CTA | `Deactivate` / `Delete permanently` (coral); cancel returns to `Keep` |

Interaction contracts:
- **Slug auto-fill (D-08):** typing the label live-fills the slug field (slugified); the slug field stays editable and shows the unique/format error inline on blur and on save.
- **"You keep" panel (D-06):** recomputes live as price or commission % changes; shows the three lines above (commission amount, net "you keep", fee note). All amounts EUR, stored as integer cents (D-07).
- **Bottom-up deactivation (D-12):** a parent's Deactivate control is disabled with the matching helper copy when it has active children.
- **Invite flow (D-01/D-04):** on submit, the console reveals the generated set-password link inline with a copy button and the delivery note; no email is sent in Phase 2 (D-03 stub).
