# Balkanity pictograms (real brand assets — D-09 / SC-5)

These SVGs are the **real** Balkanity pictograms, copied verbatim from the committed
source set under the local `Branding /` folder (which is gitignored — the masters live
on the user's machine) and renamed to normalized, lowercase, space-free filenames so
app code can import them by stable path.

- Vectors only (the `.ai` / `.png` source variants are intentionally not copied).
- Never re-drawn or invented — only the committed real marks (UI-SPEC D-09 rule).
- `transfers.svg` is the Welcome-Pickup-relevant pictogram for Phase 2+ surfaces.
